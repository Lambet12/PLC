## PLC 기말
### 사격 표적지
##### 1. 사용한 명령어 모음
- C~ 카운터 : 100M, 200M 표적지 번갈아 점등 후 250M 점등
- T~ 타이머 : 표적지 유지 시간
- RST 리셋 : 사격 이후 초기화에 사용
##### 2. 구동
1. 시작 스위치 작동
2. 100M 표적지 3초간 ON
4. 100M 표적지 OFF
5. 동시에 200M 표적지 3초간 ON
6. 반복하여 200M OFF 후 100M 다시 ON
7. 100M - 200M 점등 3번 반복 후 250M 표적지 3초간 ON
8. 전체 OFF 후 리셋
